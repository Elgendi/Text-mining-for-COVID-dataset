1. Make sure 'comm.mat', 'preprocessText.m' and 'TextMining_fullcode.m' are in the same folder.
2. Run 'TextMining_fullcode.m' in Matlab.
3. 'wordcloud_example.pdf', each populations ranking and 'ranking_table.mat' will be automatically saved to your folder.
4. The paper indices selected as output and their titles are displayed in the command window together with further information (see comments in code).

Remark: Our generated results can be found in the 'results' folder.
